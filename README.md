# Inventory Management System (PHP & MySQL)

A simple inventory management system built with PHP and MySQL.  
The project focuses on database integration and practical web development.

---

## 🚀 Features

- Add, edit, and delete inventory items  
- Store and manage data using MySQL  
- Simple and functional user interface  
- Backend logic implemented in PHP  

---

## 🛠️ Tech Stack

- PHP  
- MySQL (external MySQL Server)  
- HTML / CSS  
- Apache (XAMPP)

---

## 📦 Requirements

- XAMPP (for Apache only)  
- MySQL Server (installed separately)  
- MySQL Workbench or similar tool  
- Web browser  

---


## ⚙️ Configuration

Rename the file:

database/connexion.example.php

to:

database/connexion.php

and update it with your own database credentials.


---


## ⚙️ Installation

1. Install XAMPP  
https://www.apachefriends.org 

2. Start Apache in XAMPP and make sure your MySQL Server is running  

3. Copy the project into:

xampp/htdocs/inventory_system_management

4. Create the database `varasto` using MySQL Workbench or phpMyAdmin.

5. Import the database file:

varasto.sql

6. Open in browser:

http://localhost/inventory_system_management
